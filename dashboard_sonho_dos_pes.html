⚠️ Configuração Necessária

A tabela do banco de dados ainda não foi criada no Supabase.
Siga os passos abaixo uma única vez:

  1. Acesse supabase.com → seu projeto sonho-dos-pes
  2. Clique em SQL Editor no menu lateral
  3. Clique em New query
  4. Cole o SQL abaixo e clique em Run
CREATE TABLE IF NOT EXISTS sdp_store (
  chave TEXT PRIMARY KEY,
  valor TEXT NOT NULL,
  atualizado_em TIMESTAMPTZ DEFAULT NOW()
);
ALTER TABLE sdp_store ENABLE ROW LEVEL SECURITY;
CREATE POLICY "acesso_total" ON sdp_store
  FOR ALL USING (true) WITH CHECK (true);

Após rodar o SQL, recarregue esta página (F5).
